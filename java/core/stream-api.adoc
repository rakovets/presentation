== Intro

=== Problem

[.fragment]
Сложность решения некоторых простых задач в сравнении с функциональными языками программирования.

** Stream API

== Stream API

=== Stream API

[.step]
* Предназначен для *обработки* данных
* Не хранит данные
* Нельзя переиспользовать
* Можно преобразовать в другие типы данных
